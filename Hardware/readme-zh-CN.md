# 硬件文档 - ArmSync 7DOF 机械臂

本目录包含 ArmSync 七自由度机械臂项目的所有硬件相关文件。

## 项目简介

ArmSync 是一款采用双 IMU 主从无线遥操作控制的七自由度机械臂。本项目面向教育和研究用途，注重成本效益和易于复现。

### 主要规格

| 参数 | 数值 |
|-----------|-------|
| 自由度 | 7 (J1-J7) |
| 臂展（含夹爪） | ~60cm |
| 末端负载 | 500g（目标） |
| 控制接口 | CAN 总线 |
| 电源 | 24V 直流 |

### 关节配置

| 关节 | 名称 | 电机 | 减速比 | 功能 |
|-------|------|-------|------------|----------|
| J1 | 基座旋转 | 57×76mm 步进电机 | 20:1 | 基座旋转 |
| J2 | 肩部俯仰 | 42×60mm 步进电机 | 30:1 | 肩部上下 |
| J3 | 肩部偏航 | 42×40mm 步进电机 | 10:1 | 肩部左右 |
| J4 | 肘部 | 42×48mm 步进电机 | 10:1 | 肘部弯曲 |
| J5 | 腕部俯仰 | 42×40mm 步进电机 | 10:1 或 5:1 | 腕部上下 |
| J6 | 腕部旋转 | 42×34mm 步进电机 | - | 腕部旋转 |
| J7 | 夹爪 | MG90S 舵机 | - | 末端执行器 |

## 快速开始

### 1. 查看设计

查看 [3D Files](./3D%20Files/) 目录获取完整机械设计文件：
- **SolidWorks 2026/**: 原始可编辑源文件
- **STEP and STL/**: 通用格式，用于查看和打印
- **3MF for printing/**: 预配置的 Bambu Studio 项目文件

### 2. 采购零件

参见 [Parts BOM.md](./Parts%20BOM.md) 获取完整零件清单，包括：
- 电机规格及价格
- 减速器选型指南
- 紧固件数量
- 预估总成本（约 1400 人民币）

### 3. 打印零件

**推荐打印参数：**
- 材料：PLA Basic 或更好
- 层高：0.18mm
- 填充率：35-60%（结构件）、20%（外观件）
- 支撑：按需添加

### 4. 组装

按照 [Assembly Guide.md](./Assembly%20Guide.md) 进行分步组装：
- 组装顺序：从上到下。J1 → J2 → J3 → J4 → J5 → J6 → J7
- 紧固件扭矩规格
- 接线和 CAN 总线设置
