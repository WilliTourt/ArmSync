## 项目结构

```
Software/
├── Arm_vscode/      # 机械臂主控程序（RA6M5）
├── Ctrller_keil/    # 手部控制器程序（RA4M1）
└── IMU_keil/       # IMU模块端程序（RA4M1）
```

## 硬件平台

- **主控芯片**: Renesas RA 系列 MCU
  - RA6M5 (Arm_vscode)
  - RA4M1 (Ctrller_keil, IMU_keil)
- **开发框架**: Renesas FSP

## 功能简介

### Arm_vscode - 机械臂主控
- CAN 总线通信，控制多个舵机
- 接收手部姿态数据并解析
- 夹爪控制
- 按键中断处理

### Ctrller_keil - 手部控制器
- IMU 传感器融合（ICM42688P 六轴 + QMC5883P 三轴磁力计）
- 使用 ESKF 融合四元数
- 通过蓝牙串口发送姿态数据
- ADC 读取压力传感器控制夹爪

### IMU_keil - IMU远程端
- 同样使用 ICM42688P + QMC5883P
- 支持大臂/小臂两种模式配置
- 通过串口发送四元数数据

## 通信协议

数据包使用自定义协议，格式如下：

| 字段 | 说明 |
|------|------|
| 帧头 | 0xAF |
| 功能码 | 0x01~0x04 |
| 数据 | 四元数/欧拉角/ADC值等 |
| 帧尾 | 0xFA |
